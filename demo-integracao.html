Demo - Sistema de Captura de Dados AMV

O que foi implementado:

  • Página de captura de dados (`captura-rainha-2025.html`)
  • Formulário com validação de nome, email e WhatsApp
  • Integração com Google Sheets via Google Apps Script
  • Redirecionamento automático para as fotos após captura
  • Design responsivo e atrativo

Fluxo do usuário:

1. Usuário clica em "Ver fotos" na galeria ou página inicial
2. É redirecionado para a página de captura
3. Preenche nome, email e WhatsApp
4. Dados são validados e enviados para Google Sheets
5. Usuário é redirecionado para as fotos no Google Drive

Como configurar o Google Sheets:

IMPORTANTE: Siga as instruções no arquivo `INSTRUCOES-GOOGLE-SHEETS.md`

1. Criar a planilha:

Nome | Email | WhatsApp | Data | Página

2. Configurar Google Apps Script:

Use o código do arquivo `google-apps-script.js`

3. Atualizar URL no site:

Substitua a URL no arquivo `captura-rainha-2025.html`

Recursos implementados:

  • Validação em tempo real: Campos são validados conforme o usuário digita
  • Máscara de WhatsApp: Formato automático (54) 99999-9999
  • Feedback visual: Loading, sucesso e erro
  • Design responsivo: Funciona em mobile e desktop
  • Integração segura: Dados enviados via POST para Google Sheets

Dados capturados:

O sistema captura e armazena:

  • Nome completo do visitante
  • E-mail para contato
  • WhatsApp para comunicação
  • Data e hora da captura
  • Página de origem (Escolha da Rainha 2025)

Próximos passos:

  2. Teste o formulário de captura
  3. Monitore os dados na planilha
  4. Use os contatos para marketing e comunicação
